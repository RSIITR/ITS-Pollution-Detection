# ITS-Pollution-Detection

### Description of the folder - "yolov3"

**backup** : Folder containing weight saved dynamically after 1000 iterations while training. <br /> **obj.data**: a ".data" file containing information of the number of classes, the path to training data, and the backup folder.<br />  **obj.names**: a ".names" file consisting of class names in the correct order when matched with the training dataset's class indices.<br />  **obj.zip**: a zip file consisting of the images in .jpg format and corresponding text files with the correct bounding boxes.
