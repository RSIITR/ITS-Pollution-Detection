# ITS-Pollution-Detection

### Description of the folder - "yolov3"

backup: Folder containing weight saved dynamically after 1000 iterations while training.

obj.data: a ".data" file containing information such as the number of classes, path to training data, and the backup folder.

obj.names: a ".names" file consisting of class names in the right order when matched with indices in the training dataset.

obj.zip: a zip file consisting of the images in .jpg format, and corresponding text files with the correct bounding boxes.
